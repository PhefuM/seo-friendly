BookBuddy

BookBuddy is a fully responsive web application designed to help users discover and explore books with ease. It utilizes the Google Books API to fetch real-time data on a wide range of titles, including author information, descriptions, and cover images. Built using HTML, CSS, and JavaScript, the app delivers a smooth, mobile-friendly experience. BookBuddy features a dynamic search function, responsive layouts for all screen sizes, and a clean, user-friendly interface that encourages exploration and engagement with the world of literature.
